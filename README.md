# Blog Service API

Backend pet-проект REST API для блог-платформы, реализованный на Go с использованием Clean Architecture.
Проект включает авторизацию, управление постами, комментариями, лайками и ролями пользователей.

---

## Функциональность

- Регистрация и авторизация пользователей
- JWT (acces + refresh tokens)
- RBAC (user / admin)
- CRUD операции для постов
- Комментария к постам
- Лайки постов
- Поиск фильтрация, пагинация
- Swagger документация API
- Логирование (logrus)
- Unit и интеграционные тесты
- Docker и Docker Compose
